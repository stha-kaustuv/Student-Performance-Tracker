# 🎓 Student Performance Tracker (Frontend)

This is the frontend dashboard for the Student Performance Tracker. It provides a clean, interactive interface to visualize student data, filter by performance categories (Weak, Average, Excellent), and predict student exam scores using a Machine Learning model.

---

## 📌 Features

- Real Time Prediction
- Dynamic Data Table
- Smart Filtering

## 🚀 Tech Stack

- Framework:Vue.js 3 (Composition Api)
- Build Tool: Vite
- Styling: Tailwind CSS
- HTTP Client: Axios

---

## ▶️ How to Run the Frontend Project

### 1️⃣ Get inside the frontend Repository

```bash

cd app
cd student-performance-tracker

```

### 2.Install dependencies:

```bash
npm install
```

### 3. Configure API URL:

Ensure your Flask backend is running. Open your code and verify the Axios base URL matches your backend:

```bash
http://127.0.0.1:5000
```

### 4.Run in development mode:

```bash
npm run dev
```

### 4.Open the local host in web browser:

```bash
http://localhost:5173/
```
