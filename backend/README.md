# 🎓 Student Performance Tracker (Backend)

This is the Flask-based REST API that powers the Student Performance Tracker. It handles machine learning model predictions, serves student datasets from CSV, and provides model evaluation metadata.

---

## ▶️ How to Run the Backend Project

### 1️⃣ Get inside the frontend Repository

```bash

cd app
cd backend

```

### 2.Install dependencies:

```bash
pip install -r requirements.txt
```

### 3. Run the Server:

```bash
python app.py
```
